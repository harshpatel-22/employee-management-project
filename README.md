## Dependencies in this project are:
    1.Spring Data JPA
    2.Spring Boot Web
    3.H2 Database
    4.MySql-JDBC Driver
    5.Lombok
    6.MySql J Connector

## Methods Used:
      1.GET:  http://localhost:8080/employees
      2.POST: http://localhost:8080/employees
      3.PUT:  http://localhost:8080/employees/id
      4.DELETE: http://localhost:8080/employees/id
